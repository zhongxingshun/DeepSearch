## 五、测试过程中发现并修复的问题

| # | 问题描述 | 原因 | 修复措施 | 状态 |
|---|---------|------|---------|------|
| 1 | 缺少 greenlet 依赖 | SQLAlchemy async 需要但未安装 | `pip install greenlet` | ✅ 已修复 |
| 2 | users 表 `hashed_password` 列名不匹配 | init-db.sql 与 ORM 模型列名不一致 | ALTER TABLE RENAME COLUMN | ✅ 已修复 |
| 3 | users 表缺少 `failed_attempts`/`locked_until` 列 | init-db.sql 未包含这些字段 | ALTER TABLE ADD COLUMN | ✅ 已修复 |
| 4 | bcrypt 5.0 与 passlib 不兼容 | bcrypt 5.0 严格限制72字节 | 降级至 bcrypt==4.2.1 | ✅ 已修复 |
| 5 | 密码 hash 不兼容 | 版本切换后 hash 格式不匹配 | 用当前版本重新生成 hash | ✅ 已修复 |
| 6 | audit_logs 表列名不匹配 | `action`→`action_type` 等 | ALTER TABLE RENAME COLUMN | ✅ 已修复 |
| 7 | User 模型缺少 email 字段 | ORM 模型未声明 email | 添加 email mapped_column | ✅ 已修复 |
| 8 | Admin API 全部为 TODO stub | Sprint 5 占位代码未实现 | 完整实现所有 CRUD 接口 | ✅ 已修复 |
| 9 | 侧边栏子菜单白色文字白色背景 | el-sub-menu 默认样式覆盖 | 添加 SCSS 深色样式 | ✅ 已修复 |
| 10 | Meilisearch `documents` 索引未创建 | 未初始化搜索索引 | 手动创建索引 | ✅ 已修复 |
| 11 | tasks 表缺少 `retry_count` 列 | init-db.sql 未包含 | ALTER TABLE ADD COLUMN | ✅ 已修复 |
